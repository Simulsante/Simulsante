# Simul'Santé

Site vitrine B2B du simulateur de remboursements santé.

## Déploiement

Ce repo est connecté à Vercel. Tout push sur `main` déclenche un déploiement automatique.

## Configuration Formspree

Dans `index.html`, remplacer `VOTRE_ID_FORMSPREE` par l'identifiant obtenu sur formspree.io.

## Domaine

Domaine cible : `simulsante.fr` (à acheter sur OVH puis configurer dans Vercel).
